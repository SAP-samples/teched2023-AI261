# Create a New Model from an Excel File

To create a new model in the simulator, please import an Excel file and follow these steps:

Step 1: Firstly, you need to click on “Download Model Template.”
Note: Only excel file that are in the format of model template are accepted in the application

 <img width="397" alt="image" src="https://github.com/SAP-samples/teched2023-AI261/assets/147400890/2bc2f522-ad13-476d-b03a-f995e4951276">

 
Step 2: From the downloaded model template, you can observe the guidelines to create an excel file in expected format of the model template. 
We recommend saving ‘modeling_template.xlsx’ file as a new excel file and make changes into the new file to fit your business calculation. 

Step 3: After modifying your excel file that contains business data calculations, click on “Import an Excel file” to upload your file into the system.

<img width="397" alt="image" src="https://github.com/SAP-samples/teched2023-AI261/assets/147400890/f14a75a2-9f4e-474a-8d2a-4c2b86deba26">

Step 4: Select the correct file and click on “Upload”.

<img width="425" alt="image" src="https://github.com/SAP-samples/teched2023-AI261/assets/147400890/b3b4ea7f-b0ed-4312-8fcc-7f87aa4eaea6">

Step 5: Select the correct sheet from the file.

<img width="425" alt="image" src="https://github.com/SAP-samples/teched2023-AI261/assets/147400890/a9e57980-79cc-4c52-909f-1d47a87cd530">


Step 6: Select the required output variable cell.
  
<img width="425" alt="image" src="https://github.com/SAP-samples/teched2023-AI261/assets/147400890/5df5af7b-d689-4864-acbd-1ae9d6ccce8a">


Step 7: Choose a model name and click on Next.

<img width="425" alt="image" src="https://github.com/SAP-samples/teched2023-AI261/assets/147400890/69c3fe61-8020-41fb-9fe8-bff561792f26">

    
Step 8: Select the influencers type and click Save.
*Note: All the input cells (i.e., the cells which are without formulas) of the excel are translated into influencers with single value as default, but here you can change them to range values, as required. 

<img width="397" alt="image" src="https://github.com/SAP-samples/teched2023-AI261/assets/147400890/0b89cd8d-6950-4658-b48f-6e936c73262d">


Step 9: The Model will be opened and will also be added to the list of saved Models. 
 
<img width="397" alt="image" src="https://github.com/SAP-samples/teched2023-AI261/assets/147400890/4283f6ff-9042-4c3a-836e-73195e6e3bcb">


Step 10: When you are finished, just click “Close”.     

<img width="397" alt="image" src="https://github.com/SAP-samples/teched2023-AI261/assets/147400890/bf6ba1c2-51cd-4392-9494-e722c574565e">
